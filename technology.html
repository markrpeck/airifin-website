Security

Your Data, Your Environment

AIRIFIN is designed so that your financial data never leaves your own cloud environment.

🔒 Deployed within your GCP project

We deploy AIRIFIN inside your own Google Cloud project. Your data never moves to a shared server or AIRIFIN's infrastructure.

🔑 All credentials in Google Secret Manager

API keys, OAuth tokens, and connection strings are encrypted in Secret Manager — never in code, never in logs.

📋 Every action is logged

Full audit trail of every query, sync, and report creation. Accessible from your GCP project at any time.

🏥 BAA available for healthcare clients

Google Cloud BAA covers the infrastructure. Anthropic BAA available for clinical AI use cases. HIPAA-eligible architecture by design.
